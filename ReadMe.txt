ExcelVBA用マクロ集 各フォルダの説明

《FileNameConvert》フォルダ
ファイル名・フォルダ名を取得、変換するツール

《MergedCellsRowAutoFit》フォルダ
結合されたセルの行の高さを自動調整するマクロ

《FileDownload》フォルダ
指定したURLのファイルを一括ダウンロードするマクロ

《AutofilterExcludeMultiple》フォルダ
オートフィルターで特定の列の複数項目を除外するマクロ

《SlicerControl》フォルダ
データソースの異なるピボットテーブルのスライサーと連動させるマクロ

《PrefNameComplement》フォルダ
都道府県の抜けている住所から、都道府県・市区町村名を取得するツール

《DynamicChartRange》フォルダ
グラフの情報を読み取り、グラフのデータ範囲を自動的に変更する設定を行うマクロ

《CreateScheduleCalendar》フォルダ
予定一覧から予定管理用カレンダーを作成するマクロ

《ApplySameFormatCharts》フォルダ
選択したグラフの書式・大きさにグラフを統一するマクロ

《CreateMailTool》フォルダ
エクセルの宛先・本文データからデータを差し込んだメールを作成するマクロ
